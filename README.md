# Ericeira-WeathrePro
WeatherPro é um sistema de meteorologia que agrega dados de várias APIs e serviços (IPMA, AccuWeather, entre outros), calcula médias inteligentes e utiliza IA para gerar um parecer final sobre o estado do tempo. Este repositório contém apenas a versão pública do frontend; toda a lógica interna, APIs, scrapers e backend permanecem privados.
