🌦️ WeatherPro —

WeatherPro é uma plataforma de meteorologia avançada que reúne dados de várias fontes oficiais e comerciais — como o IPMA, AccuWeather e outros serviços internacionais.
O sistema cruza e valida a informação, calcula uma média inteligente e utiliza IA para interpretar tendências, gerar alertas e apresentar um parecer final claro e fiável sobre o estado do tempo.

🔗 Versão online: https://alvarofr.pt/weather/  
📱 APK (Beta na Play Store): https://play.google.com/apps/internaltest/4701756317798342924

📁 Este repositório contém apenas os ficheiros públicos do frontend.  
🔒 Toda a lógica interna, APIs, scrapers e backend permanecem privados por motivos de segurança.rança.
